gaia.core.hierarchical_messaging module

Complete Hierarchical Message Passing for GAIA Framework

Implements Section 3.4 from GAIA paper: Hierarchical Message Passing

THEORETICAL FOUNDATIONS: - _ parameters for each simplex X_n - Local objective functions L_(_{d_0},,_{d_n}) for each simplex - Update rule combining gradient information from (n+1) faces - Instructions from degeneracies that include as face - Inner horn solvers update parameters by composing gradients from face maps - Outer horn solvers train inverses - Hierarchical update scheme where information percolates up/down simplicial complex
